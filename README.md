# 조선 야행 (Joseon Night Walk)

칠흑 같은 조선 산속, 등불 하나에 의지해 미로를 탈출하는 Three.js 1인칭 공포 게임.
컴퓨터그래픽스 기말 프로젝트 — 핵심 기술로 **DDGI**(Dynamic Diffuse Global Illumination)를 직접 구현한다.

## 조작
- `WASD` 이동 · 마우스 시점 · `Shift` 달리기
- 화면 클릭 → 시작(포인터 잠금)

## 개발

```bash
npm install
npm run dev      # http://localhost:5180
npm run build    # dist/ 정적 빌드
npm run preview  # 빌드 미리보기
```
